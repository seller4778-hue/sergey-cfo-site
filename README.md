# Сайт Сергея Чередниченко

Это статический сайт-визитка: HTML + CSS.

## Как посмотреть локально
Откройте файл `index.html` двойным кликом в браузере.

## Как заменить фото
1. Положите свое фото в папку `assets`.
2. Назовите файл `photo.png`.
3. Обновите страницу.

## Как выложить бесплатно
Вариант 1: GitHub Pages
- Создать репозиторий на GitHub.
- Загрузить файлы `index.html`, `styles.css` и папку `assets`.
- Включить Settings → Pages → Deploy from branch → main.

Вариант 2: Vercel
- Создать репозиторий на GitHub.
- Импортировать репозиторий в Vercel.
- Нажать Deploy.

## Важно
Форма заявки сейчас работает через `mailto:` — она открывает почтовую программу пользователя.
Позже можно подключить Google Forms, Formspree, Tally или backend-форму.
